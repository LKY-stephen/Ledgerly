# 记账 App 功能梳理（参考：喵喵记账 / 鲨鱼记账）

> 基于参考记账app归纳，用于产品规划与自研对照。结构：**先完整梳理喵喵记账 → 再完整梳理鲨鱼记账 → 最后对比**。

---

# 一、喵喵记账

## 1.1 产品概览与信息架构

- **视觉与调性**：手绘、猫咪主题，暖色与 paw 底纹，空状态与提示语带「喵」等拟人化。
- **主导航（常见三 Tab）**  
  - **明细/日历**：流水列表、月维度、搜索与新建。  
  - **统计**：当月收支概况、图表向展示。  
  - **资产**：净资产、资产账户列表与趋势。  
- **其他导航变体**：部分版本底部为 **明细、图表、+、社区、我的**（与三 Tab 版可能为不同版本或子模块）。

## 1.2 登录注册与账号（截图无独立页，从「我的」与行业惯例推断）

- 用户展示为 **昵称 + 可编辑**，如「喵喵61955493」，暗示 **游客/快速进入后可绑定** 或 **手机号/三方登录**。
- **设置**（齿轮）、**门图标**（部分页左上）可能承担退出、返回或切换场景。
- 若产品上云：**手机号验证码、微信/QQ/Apple 登录、游客绑定合并、隐私协议** 等与其它记账 App 对齐即可；喵喵侧运营位多，账号常与 **会员、新人福利** 联动。

## 1.3 记账（记一笔）

- **支出 / 收入**：顶部 Tab 切换。  
- **聊天记账**：与支出、收入并列的入口，对话式或语句解析记账（若自研需单独拆：解析、纠错、确认单）。  
- **分类**：圆形图标网格，多页圆点分页；示例类：餐饮、购物、日用、交通、蔬菜、水果、零食、运动、娱乐、通讯、服饰、美容、住房、家庭、社交、旅行等。  
- **账本**：顶栏 **默认账本**，支持多账本语境下的当前账本选择。  
- **金额**：自定义数字键盘；**+ / −** 简易运算；**今天** 改日期；**完成** 保存。  
- **备注**：「点击输入备注…」。  
- **账户**：左侧钱包图标，选资金账户（与资产联动）。  
- **附件**：金额区旁 **相机**，拍照挂账单。

## 1.4 明细 / 日历（首页）

- 标题：**某年某月账目明细**；时钟类入口（历史/日历）。  
- **账本下拉**：默认账本。  
- **自动记账**：如「自动记账(未开启)」及开启入口。  
- **月度卡片**：本月结余、本月支出、本月收入；**设置预算**。  
- **搜索**：账单 / 备注 / 分类等。  
- **新建**：「+ 新建」。  
- **空状态**：猫咪插画 + 文案（如「抱歉当前时间没有记录哦喵~」）。  
- 另一版式：顶栏 **喵喵记账** + 当月支出/收入汇总，引导点底部「+」记账。

## 1.5 统计

- **某年某月收支概况**；**支出 / 收入** 切换。  
- 门/时钟类导航习惯与明细一致。  
- **空状态**：插画 +「去记账!」强 CTA。

## 1.6 资产

- **净资产**（总资产 − 负债），**小眼睛** 显隐。  
- 总资产、负债；**查看资产趋势**。  
- **资产账户**：区块标题 + **+ 添加**；空状态插画与引导（如「还未添加账户哦喵~」）。  
- 区块旁 **帮助（？）**。

## 1.7 社区与运营

- **社区** Tab（部分版本主导航）。  
- **我的** 内：**立即领取新人福利**、**喵喵粉钻卡/会员**、**天天领好券**、连续记账奖励等运营入口。

## 1.8 我的（个人中心）

- **头像、昵称**（可编辑）；**记账天数、记账笔数**。  
- 快捷入口：**账本、小组件（NEW）、个性化（NEW）**。  
- 宫格功能：**定时记账、密码锁、账单导出、账单导入、自动记账、购物清单、记账提醒、更多**。  
- 底部列表：会员、优惠券、福利等导流。

# 二、鲨鱼记账

## 2.1 产品概览与信息架构

- **视觉与调性**：黄白主色、鲨鱼 IP，工具感强、信息密度高。  
- **主导航（五 Tab）**  
  - **明细**：月筛选、汇总、流水列表与快捷入口。  
  - **图表**：周/月/年、趋势与分类排行。  
  - **记账**：中央大号 **「+」**，全局强 CTA。  
  - **发现**：账单/预算/资产摘要 + 常用工具矩阵。  
  - **我的**：资料、VIP、账本、家庭、安全、设置等。

## 2.2 登录注册与账号（截图无独立页，从「我的」推断）

- 用户展示如 **鲨鱼643537237**，同样符合 **游客或弱账号 + 可绑定手机/三方** 的常见设计。  
- **账户安全中心**：绑定、密码、设备与风险提示（如「高风险」引导绑手机）。  
- **设置** 双入口（顶栏快捷 + 列表）较常见。

## 2.3 记账（记一笔）

- **支出 / 收入**：顶栏 Tab；右上角 **取消** 退出。  
- **分类**：黄头 + 图标网格；除餐饮、购物、日用、交通等基础类外，截图可见更细类：**居家、孩子、长辈、烟酒、数码、汽车、医疗、书籍、学习、宠物、礼金、礼物、办公** 及维修/其他向分类。  
- 进入金额、备注、日期、账户等步骤与行业通用流程一致（与喵喵同属「选类 → 填金额」路径，具体步序以实际 App 为准）。

## 2.4 明细（首页）

- **年月** 下拉；当月 **收入、支出、结余**。  
- **快捷入口**：账单、预算、资产管家、购物返现、更多等。  
- **列表区**：有数据时按日聚合；空状态「还没有记录哦」「暂无数据」等。  
- 另一截图变体：顶栏 **鲨鱼记账** + 月汇总 + 快捷图标行。

## 2.5 图表

- 类型：**支出**（下拉可切收入等）。  
- 时间：**周 / 月 / 年**。  
- 指标：**总支出、平均值**（随周期变）。  
- **折线趋势图**（按日横轴）。  
- **支出排行榜**：按分类排序，无数据占位。

## 2.6 发现

- **账单**：当月收入、支出、结余摘要。  
- **总预算**：环形进度、剩余/本月预算、本月支出；**设置预算**。  
- **资产管家**：净资产、资产、负债摘要，点击进入管理。  
- **常用功能**：省钱优惠券、资产管家、发票助手、房贷计算器、汇率换算器等。

## 2.7 资产与预算（与「我的」交叉）

- **发现** 与 **我的** 均可能进入 **资产管理、我的钱包、预算设置**。  
- 资产维度与喵喵一致：**净资产 = 资产 − 负债**，多账户维护。

## 2.8 我的（个人中心）

- **打卡**；统计项含 **连续打卡、记账总天数、记账总笔数**（不同版本文案可能微调）。  
- **升级为 VIP** 等会员卡片。  
- 顶栏快捷：**消息（角标）、徽章、积分、邀请好友（领会员）、设置（角标）**。  
- 列表：**我的账本、家庭账单**；**设置、账户安全中心、使用帮助、意见反馈、评分、关于** 等。  
- 另一版布局：**宫格**（徽章、存钱计划、记账提醒、账单导出、定时记账、小组件、汇率换算、更多）+ **列表**（我的钱包、预算设置、资产管理、账单导入、密码锁、皮肤中心、反馈、关于）。

# 三、喵喵 vs 鲨鱼 — 对比

## 3.1 定位与体验

| 维度 | 喵喵记账 | 鲨鱼记账 |
|------|----------|----------|
| 视觉与品牌 | 手绘猫咪、软萌、文案拟人 | 鲨鱼 IP、黄白工具风 |
| 主导航 | 以 **明细/统计/资产** 三 Tab 为主（另有五 Tab + 社区变体） | **五 Tab**，**中央 +** 强化记账 |
| 记账入口 | 明细页「+ 新建」、统计空状态「去记账!」等 | **全局中央 +**，从各主 Tab 一键进入 |
| 差异化卖点 | **聊天记账、自动记账、购物清单**；新人福利/粉钻/券 | **发现页** 聚合账单/预算/资产/工具；**家庭账单、徽章积分、打卡、存钱计划** |
| 分类广度 | 截图以常见生活类为主，分页扩展 | 默认分类更细（居家、孩子、医疗、宠物等） |

## 3.2 模块对照（简要）

| 模块 | 喵喵记账 | 鲨鱼记账 |
|------|----------|----------|
| 明细/流水 | ✓ 月维度、搜索、自动记账入口、预算入口 | ✓ 月筛选、快捷入口条、购物返现等 |
| 统计/图表 | 统计 Tab，月概况 + 支出收入切换 | 图表 Tab，周月季 + 趋势 + 分类排行 |
| 资产 | 独立 **资产** Tab，净资产显隐、趋势、账户 | **发现** 摘要 + **我的** 资产管理/钱包 |
| 预算 | 明细卡片内「设置预算」 | **发现** 总预算卡片 + **我的** 预算设置 |
| 社区/发现 | **社区** Tab + 我的内运营 | **发现** Tab 工具与导流 |
| 我的 | 账本/小组件/个性化；宫格偏 **导入导出、自动记账、购物清单** | 打卡/VIP/消息/积分/邀请；**家庭账单、皮肤中心、存钱计划** 等 |

## 3.3 登录注册（两 App 共性归纳）

- **游客或弱账号起步** + **昵称 ID** 展示，后续 **绑手机 / 三方** 较普遍。  
- **账户安全中心** 在鲨鱼「我的」中更显性；喵喵侧可放在设置或安全子页。  
- 自研建议单独 PRD：**注册登录、绑定、找回、注销与数据、多设备同步、隐私同意**。

## 3.4 横切能力（两款均常见，便于自研一张表）

| 能力 | 喵喵侧重 | 鲨鱼侧重 |
|------|----------|----------|
| 多账本 | 顶栏默认账本、我的-账本 | 我的账本、家庭账单 |
| 预算 | 明细卡片入口 | 发现总预算 + 我的 |
| 定时/周期记账 | ✓ | ✓ |
| 记账提醒 | ✓ | ✓ |
| 导入/导出 | ✓ | ✓ |
| 密码锁 | ✓ | ✓ |
| 小组件 | ✓（NEW 标注） | ✓ |
| 皮肤/主题 | 个性化 | 皮肤中心 |
| 会员商业化 | 粉钻、新人福利、券 | VIP、邀请领会员、积分 |
